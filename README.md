# Car-Price-Prediction-RF

**DESCRIPTION**

Predicting the car price for the customers using the variables like
a. seller_type
b. Fuel_type
c. Transmission Owner
d. Car_name
e. present_price
f. selling_price
h. kms_driven

**Feature Engineering**
a. One hot encoding
b. adding the number of years column

**MODEL**
a. Random Forest

**Performance Metrics**
a. MAE
b. MSE
c. RMSE
